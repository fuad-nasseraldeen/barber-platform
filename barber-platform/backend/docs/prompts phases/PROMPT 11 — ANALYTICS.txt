Analytics Implementation
Endpoint
Method	Endpoint	Auth	Description
GET	/api/v1/analytics?businessId=&startDate=&endDate=	JWT + RBAC	Returns all analytics metrics
Query Parameters
businessId (required) – business UUID
startDate (optional) – start of period (YYYY-MM-DD)
endDate (optional) – end of period (YYYY-MM-DD). Default: today. If omitted, last 30 days are used.
Metrics
Metric	Description
revenueByStaff	Revenue and booking count per staff (from SUCCEEDED payments)
revenueByService	Revenue and booking count per service
customerRetention	totalCustomers, repeatCustomers, retentionRate, newCustomers
busyHours	Appointment count per hour (0–23)
dailyBookings	Appointment count per date
staffPerformance	Per staff: totalBookings, completedBookings, cancelledBookings, revenue, completionRate
